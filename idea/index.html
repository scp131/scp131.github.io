maven配置文件配置

#配置本地仓库文件夹
<localRepository>F:\Maven_respo</localRepository>


#配置阿里云镜像
<mirrors>
    <mirror>
        <id>nexus-aliyun</id>
        <mirrorOf>central</mirrorOf>
        <name>Nexus aliyun</name>
        <url>http://maven.aliyun.com/nexus/content/groups/public</url>
    </mirror>
<mirrors>
    
    
#配置java版本
<profile>
<id>jdk-1.8</id>
<activation>
	<activeByDefault>true</activeByDefault>
	<jdk>1.8</jdk>
</activation>
<properties>
	<maven.compiler.source>1.8</maven.compiler.source>
	<maven.compiler.target>1.8</maven.compiler.target>
	<maven.compiler.compilerVersion>1.8</maven.compiler.compilerVersion>
</properties>
</profile>

修改junit版本为4.11(pom.xml 3.x不支持注解式测试)

<dependency>
  <groupId>junit</groupId>
  <artifactId>junit</artifactId>
  <version>4.11</version>
  <scope>test</scope>
</dependency>

常用快捷键

Ctrl + Alt + V    快速生成返回值

Alt + Insert    Set/Get 重写 toString Constructor ……

Ctrl + Z / Ctrl + Shift + Z    撤销 / 前进

Shift + Ctrl + ↑ / Shift + Ctrl +↓    代码块向上/向下移动

Ctrl + Y    删除一行代码


Ctrl + N    查找类

Shift + Ctrl + N    查找文件

Ctrl + P    方法参数提示

Ctrl + Space    代码提示


Ctrl + Alt + O    优化导入类和包

Ctrl + Alt + U    向上查看继承结构

Ctrl + H    打开查看类控制版面
