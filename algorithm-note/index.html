滑动窗口算法

将首尾指针内的数据看作一个队列（窗口），这个队列随着指针的变化在不断的移动，即滑动窗口。

前提

  • 需要首尾指针
  • 首尾指针在不断的移动
  • 首尾指针之内的内容为目标数据
  • 目标数据存储在辅助容器中

示例

//寻找字符串中最长的字串
//字串为首尾指针之间的字符串
int start = 0;
int end = 0;
int max = 0;
Map<Character,Integer> map = new HashMap<>();
for(;end < str.length(); end++){
    char temp = str.charAt(end);
    if(map.contatinKey(temp)){
        start = Math.max(max,map.get(temp));
    }
    max = Math.max(max,end - start + 1);
    map.put(temp,end + 1);
}
